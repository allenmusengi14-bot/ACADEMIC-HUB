# UB Academic Hub

A comprehensive academic platform for University of Botswana students to track GPA, find study resources, connect with tutors, and earn points through academic achievements.

## Features

### 📊 GPA Tracker
- Calculate term GPA and project CGPA
- Track module grades and credits
- Exam mark forecaster - calculate what you need on final exams
- Save and load GPA data locally

### 🎓 Study Hub
- Find resources for your modules
- Upload course outlines
- Search for videos, articles, and quizzes

### 🔥 Study Streak
- Pomodoro timer with module selection
- Stopwatch with lap functionality
- Heat calendar showing study activity
- Track study streaks and earn points
- Log study sessions manually

### 🤝 PeerBridge
- Post help requests with custom pricing
- Browse open requests from other students
- Offer help as a tutor
- WhatsApp integration for direct communication

### ⭐ Find a Tutor
- Browse peer tutors by faculty and subject
- Rate tutors and leave feedback
- Contact tutors via WhatsApp

### 👥 Study Squad
- Create and join study sessions
- Create permanent study groups
- Group chat functionality
- Filter by faculty and search

### 👥 Focus Rooms
- Create focus rooms with duration limits
- Daily minimum study requirements
- Real-time timers synced with room
- Leaderboard for room participants
- Earn bonus points for group study

### ⭐ Earn Points
- Submit test scores to earn points
- Faculty-specific leaderboard
- Monthly reset
- Point system based on marks:
  - 40-49%: 5 points
  - 50-59%: 10 points
  - 60-69%: 15 points
  - 70-79%: 20 points
  - 80-89%: 25 points
  - 90-100%: 30 points

## Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Backend**: Firebase Firestore
- **Hosting**: GitHub Pages
- **Fonts**: Google Fonts (Fraunces, Plus Jakarta Sans)
